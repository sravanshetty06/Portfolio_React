# Sravan Doniparthi - Portfolio

Welcome to my professional portfolio website! This is a modern, responsive portfolio built with React, TypeScript, and Vite.

## About

This portfolio showcases my skills, projects, and experience in web development and software engineering. It features a sleek, modern design with smooth animations and an intuitive user interface.

## Features

- **Responsive Design**: Fully responsive and works seamlessly across all devices
- **Modern UI**: Built with Tailwind CSS for a clean, professional appearance
- **Smooth Animations**: Framer Motion provides fluid, engaging animations
- **Component-Based**: Modular React components for maintainability
- **TypeScript**: Fully typed for better code quality and developer experience

## Tech Stack

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Font**: Inter from Google Fonts

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Sravan_Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

   The app will be available at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

This generates an optimized build in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```
